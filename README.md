# Wordbook

Wordbook is a native macOS English dictionary app built with SwiftUI. It is menu-bar first, with a main dictionary window, favorites, recents, Word of the Day, pronunciation playback, and settings.

## Development

Run tests:

```bash
swift test
```

Build and launch the app bundle:

```bash
./script/build_and_run.sh
```

Verify launch:

```bash
./script/build_and_run.sh --verify
```
